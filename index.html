As tags HTML são essenciais para estruturar o conteúdo de uma página web. Elas definem diferentes tipos de elementos, como cabeçalhos, parágrafos, listas, imagens, links e formulários. Lembre-se de que o uso correto das tags HTML não apenas estrutura o conteúdo, mas também melhora a experiência do usuário e a visibilidade nos mecanismos de busca.

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Com o HTML, você pode criar seu próprio site.

Na W3Schools, você encontrará referências completas sobre elementos HTML, atributos, eventos, cores, conjuntos de caracteres, codificação de URLs, suporte de navegadores e muito mais.

Tags de Cabeçalho

As tags HTML que usamos para criar títulos em páginas web são: h1, h2, h3, h4, h5 e h6 . O significado dessas tags é o mesmo, que é definir títulos, mas o nível de importância é diferente para cada uma, onde h1 tem o nível mais alto e h6 o mais baixo.

A montagem HTML dos títulos afeta diretamente o SEO da página web e acessibilidade. Por exemplo, o Google utiliza a tag h1 para detectar o título principal.

Se você tem um site e se preocupa com SEO, acessibilidade e experiência do usuário, a criação de bons títulos é fundamental, siga as dicas a seguir:

  • Use apenas uma tag h1 por página, mesmo sendo permitido mais de um.
  • Não pule os níveis de títulos. Use h1 depois h2
  • Não use tags de título para palavras sem essa importância
  • Em tags section utilize título h2 ou menor

Tags de Listas

Existem três tipos principais de listas em HTML:

  1. Listas Ordenadas (OL - Ordered Lists):
    • As listas ordenadas são utilizadas para mostrar uma sequência lógica de itens, como uma lista numerada.
    • Os itens são listados de forma ordenada, seja por número, letra ou algarismos romanos.
    • Para criar uma lista ordenada, utilize a tag ol, e os itens da lista ficam dentro da tag li.
    • Também é possível ordenar por letras maiúsculas usando ol type="a".
  2. Listas Não Ordenadas (UL - Unordered Lists):
    • As listas não ordenadas são usadas para apresentar itens sem uma ordem específica, como uma lista com marcadores.
    • Iniciam com a tag ul e os itens ficam dentro da tag li.
  3. Listas de Definição (DL - Description Lists):
    • As listas de definição são usadas para definir termos e seus significados.
    • São representadas pelas tags dl, dt (termo) e dd (descrição).
Em resumo, as listas desempenham um papel importante na estruturação de conteúdo em páginas HTML, seja para listar itens ou definir termos e suas explicações.

Tags de Formatação de Texto

Vamos explorar algumas tags de formatação avançada em HTML que podem ser úteis para marcar recursos semânticos mais específicos:

  1. p: A tag p é usada para criar parágrafos de texto.
    Ela envolve um bloco de texto e adiciona espaçamento antes e depois do conteúdo
  2. Mark (Destaque de Texto)
    • O elemento mark representa um trecho de destaque em um texto, como uma sequência de palavras marcada como referência ou relevante em um contexto específico.
  3. Strong (Ênfase Forte)
    • O elemento Strong é usado para denotar trechos de importância especial, como palavras-chave, títulos ou informações críticas.
    • Ele geralmente é exibido em negrito.
  4. Blockquote

    (Citação de Bloco)

    • O elemento Blockquote indica que o texto incluído é uma longa citação.
    • Normalmente, é processado visualmente com recuo.
    • A URL da fonte da citação pode ser especificada usando o atributo Cite, enquanto uma representação textual da fonte pode ser dada usando o elemento Cite.

Tags de Ligação

    Essas tags são essenciais para criar conexões entre diferentes partes de uma página web. Aqui estão algumas das principais:
  1. a (Âncora ou Link):
    • A tag a é usada para criar links entre páginas ou recursos.
    • Ela pode apontar para URLs, endereços de e-mail, arquivos locais e âncoras dentro da mesma página.
  2. link (Ligação Externa):
    • A tag link é usada no head para vincular recursos externos, como folhas de estilo CSS.
  3. area (Área de Imagem):
    • A tag area é usada em conjunto com a tag map para criar áreas clicáveis em imagens.
    • Útil para mapas de imagem ou gráficos interativos.
  4. iframe (Quadro Incorporado):
    • A tag iframe permite incorporar conteúdo de outras páginas dentro da sua.
    • Útil para incorporar vídeos, mapas ou widgets.
  5. base(Base URL):
    • A tag base define a URL base para todos os links relativos em uma página.
    • Especifica o diretório ou domínio padrão para os links.

Tags Pouco Usadas

  1. font: A tag font era usada antigamente para definir o tamanho, a cor e a fonte do texto. No entanto, ela está obsoleta e não é mais recomendada para uso. Em vez disso, é preferível usar CSS para estilizar o texto. Por exemplo, em CSS, você pode definir a cor do texto usando color: red;.
  2. del: A tag del é usada para indicar texto que foi removido ou riscado. Ela é frequentemente usada em documentos revisados ou em sistemas de controle de versão.
  3. abbr : A tag abbr representa uma abreviação e opcionalmente fornece uma descrição completa para ela. O atributo title deve conter a descrição completa da abreviação.
